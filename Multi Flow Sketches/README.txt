Steps for execution:
--------------------
1. Once the zip file is extracted scoure code can be found inside src folder

2. Open terminal and cd to .../src directory

3. In terminal enter the following commands:
	a) javac MultiFlowSketches.java
	b) java MultiFlowSketches

4. You will prompted to enter value for m, l and values for bSkt which are m, l, k.

5. Once the execution completes following three files will be generated:
	a) output.txt -- Report for Virtual BitMap
	
Report Description
------------------

1. output.txt:
	Contains 25 lines, one for each top-25 flow, in the descending order of their estimated spread values. With each line starts with flow id followed by estimated spread.
